Before being able to run main.ino the "TinyGPSPlus.zip" library must be imported. The library to be imported is included in the repository.

Otherwise, follow schematic and pin definitions as defined in the main.ino. Some pins may be slightly changed from the schematic, follow as described in main.ino. 

Note that new_main contains code for the unfinalized sleep implementation.


"BlindCare" folder is the app code, which was made in Swift using Xcode.
